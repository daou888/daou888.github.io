# rypto-clipper

Features
Multiple cryptos supported (BTC, LTC, XMR, ETH, XRP, NEO, BCH, DOGE, DASH, TRON, BNB, ADA, DOT)
Almost fully undetected (working on that)
Great performance (<1% CPU)
Autostart after reboot
Detect addresses inside sentences
